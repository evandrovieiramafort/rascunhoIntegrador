pra rodar esse gerador de uml:

- baixar extensão chamada "PlantUml".
- inserir o código dentro de um arquivo .puml
- apertar alt + d pra gerar o diagrama.
- pra salvar o diagrama: entra no arquivo .puml, aperta ctrl + shift + p, escolhe "PlantUml: Export Current Diagram" e escolha o formato do diagrama.
